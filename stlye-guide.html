Interactive Elements

In-text link

CSS selector:

.link

Sample code:

<p>This is an example paragraph with an <a href="#" class="link">in-text link</a>.</p>

Rendered element:

This is an example paragraph with an in-text link.

Navigation Link

CSS selector:

.nav-button

Sample code:

<a href="#" class="nav-button">Default Button</a>

Rendered element:

Default Button

Submit/Reset Buttons

CSS selector:

.nav-button

Sample code:

<input class="nav-button form-button" type="submit" value="Submit">
<input class="nav-button" type="reset" value="Reset">

Rendered element:

Text Input Field with Label

CSS selector:

label[for="input-text"]

#input-text

Sample code:

<label for="input-text">Label:</label>
<input type="text" name="input-text" id="input-text" placeholder="Placeholder Text">

Rendered element:

Drop-down List with Label

CSS selector:

#option1

label[for="option1"]

#option2

label[for="2"]

Sample code:

<form class="form-payment">
<label for="cc-type">Credit Card:</label>
<select name="cc-type" id="cc-type">
<option value="MC">Mastercard</option>
<option value="V">Visa</option>
<option value="AMEX">American Express</option>
<option value="O">Other</option>
</select> </form>

Rendered element:

Image as a Link

CSS selector:

#logo

Sample code:

<a href="#"><img src="img/newegg_ca.jpeg" class="logo" alt="Newegg Canada"></a>

Rendered element:
